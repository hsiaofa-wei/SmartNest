# NestAI Rent · 智能租房平台

全栈智能租房系统：基于 **Spring Boot + Vue 3** 前后端分离架构，集成高德地图、支付宝支付、AI 智能客服、Redis 性能优化，支持房东发布、租户预约、在线支付、角色权限的全流程闭环。

## 项目简介
NestAI Rent 是一个现代化智能租房平台，服务于房东、租户、管理员三类用户，提供从房屋发布 → 智能推荐 → 预约看房 → 在线支付 → 订单管理的完整租赁体验。

## 核心亮点：
- **LBS + AI** 深度融合：高德地图选点/聚合 + DeepSeek AI 智能客服
- **支付闭环**：支付宝沙箱模拟真实支付流程
- **性能优化**：Redis 缓存热点数据，核心接口响应时间降低 **40%**
- **多角色权限**：管理员审核、房东管理、租户预约
- **实用扩展**：天气小部件、AI 聊天助手

项目独立开发，深度使用阿里生态（高德 + 支付宝 + 大模型），适合后端/全栈/AI 应用实习展示。

## 技术栈

### 后端
- Spring Boot 2.7.14 + Java 8
- Spring Security + JWT（权限认证）
- Spring Data JPA（ORM）
- MySQL + Redis（缓存）
- 支付宝 SDK（支付集成）
- Hutool + Lombok
- Maven

### 前端
- Vue 3 + Element Plus
- Vue Router + Vuex
- Axios（API 请求）
- ECharts（数据可视化）
- Three.js（3D 效果，可选）
- 高德地图 JS API

## 核心功能
- 房屋管理：发布/编辑/审核/列表/详情/图片上传
- 租赁流程：预约看房 → 创建订单 → 在线支付 → 订单跟踪/退款
- 智能推荐：基于位置/价格/偏好的房屋推荐
- AI 客服：DeepSeek 大模型智能问答
- 地图交互：高德地图选点、聚合标记、周边天气联动
- 用户角色：管理员审核、房东发布、租户预约/收藏
- 数据统计：ECharts 看板（订单/房屋/用户统计）

## 项目亮点
- 完整租赁闭环 + 支付集成（支付宝沙箱）
- AI + LBS 深度融合（智能客服 + 高德地图）
- Redis 缓存优化，响应速度显著提升
- 多角色 RBAC 权限控制（Spring Security + JWT）
- 响应式设计 + 流畅交互体验
- 代码模块化、注释详细、结构清晰
